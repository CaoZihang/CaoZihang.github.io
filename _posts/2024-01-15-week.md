# 业界动态

## [国家数据局等17部门联合印发《“数据要素×”三年行动计划（2024—2026年）》](https://mp.weixin.qq.com/s/utNOAXwc_NHEXQdNqMZo7A)

为发挥数据要素乘数效应（报酬递增、低成本复用），17部门联合发布《行动计划》，以推进数据要素协同优化、复用增效、融合创新作用发挥为中点，强化场景需求牵引，带动数据要素高质量供给、合规高效流通，培育新产业、新模式、新动能，充分实现数据要素价值。

未来三年行动的四方面基本原则：

- 需求牵引、注重实效；
- 试点先行、重点突破；
- 有效市场、有为政府；
- 开放融合、安全有序

未来三年选取的重点行业为：工业制造、现代农业、商贸流通、交通运输、金融服务、科技创新、文化旅游、医疗健康、应急管理、气象服务、城市治理、绿色低碳

**Strengths**

我国数字经济快速发展，数字基础设施规模能级大幅跃升，数字技术和产业体系日臻成熟

**Weaknesses**

数据供给质量不高、流通机制不畅、应用潜力释放不够

**Opportunities**

我国超大规模市场、海量数据资源、丰富应用场景

**Threats**

——

**战略方向：SO战略**

推动数据要素与劳动力、资本要素协同，以数据流引领技术流、资金流、人财流、物资流，突破传统资源要素约束，提高全要素生产率

促进数据多场景应用、多主体复用，培育基于数据要素的新产品和新服务，实现知识扩散、价值倍增，开辟经济增长新空间

加快多元数据融合，以数据规模扩展和数据类型丰富，促进生产工具创新升级，催生新产业、新模式，培育经济发展新功能

**总体目标：DDL-2026.12.31**

- 300个以上示范性强、显示度高、带动性广的经典应用场景
- 涌现一批成效明显的数据要素应用示范地区
- 培育一批创新能力强、成长性好的数据商和第三方专业服务机构
- 形成相对完善的数据产业生态，数据产品和服务质量效益明显提升，数据产业年均增速超过20%
- 场内交易与场外交易协调房展，数据交易规模倍增，推动数据要素价值创造的新业态成为经济增长新动力

**操作**

  - 创新研发模式、推动协同制造、提升服务能力、强化区域联动、开发使能技术

  - 提升多式联运效能、推进航运贸易便利化、提升航运服务能力、挖掘数据复用价值、推进智能网联汽车创新发展
- 数据要素×金融服务
  - 提升金融服务水平、提升金融抗风险能力
- 数据要素×科技创新
  - 推动科学数据有序开放共享、以科学数据助力前沿研究、以科学数据支撑技术创新、以科学数据支持大模型开发、探索科研新范式
- 数据要素×文化旅游
  - 培育文化创意新产品、挖掘文化数据价值、提升文物保护利用水平、提升旅游服务水平、提升旅游治理能力

## [python丨PEP 736 – Shorthand syntax for keyword arguments at invocation关键字参数语法糖]( https://peps.python.org/pep-0736/)

 python支持简写关键字参数的语法糖了，感觉很实用！

```python
my_function(
  my_first_variable=my_first_variable,
  my_second_variable=my_second_variable,
  my_third_variable=my_third_variable,
)
```

现在可以简写为：

```python
my_function(my_first_variable=, my_second_variable=, my_third_variable=)
```
