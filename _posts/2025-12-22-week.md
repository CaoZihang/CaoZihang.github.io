## How Perceptual Disfluency Affects Consumer Choices [^1]

![20251208231650-2025-12-08-23-16-51](https://img.caozihang.com/img/20251208231650-2025-12-08-23-16-51.png)

{:.academia}
| 主题 | Title                                              | Author                                 | Journal | ROF                                                          |
| ---- | -------------------------------------------------- | -------------------------------------- | ------- | ------------------------------------------------------------ |
| 决策 | How perceptual disfluency affects consumer choices | Shahryar Mohsenin; Kurt P Munz【2025】 | JCR     | 进行消费选择时，感知不畅可能导致消费者更依赖直观信息，如品牌知名度、推荐等，而非需要仔细考虑的产品规格数字；当消费者追求决策速度时，这种效应会被放大；该效应至少部分源自消费者将信息感知不畅错误地归因为决策复杂，当消费者被告知决策不畅的实际来源（如字体）或不进行消费决策（如仅排序）时，该效应会消失 |

{:.academia}
| SPL                                                          | CPL                                                          | GAP                                                      | RFW                                                          |
| ------------------------------------------------------------ | ------------------------------------------------------------ | -------------------------------------------------------- | ------------------------------------------------------------ |
| 对某些非选择任务，如逻辑推理，感知不畅可以作为异常信号，提高警觉并激发深层处理；部分研究认为感知不畅性会导致消费者更多关注分析型线索，提升多样性寻求，另一部分认为不存在该效应；感知不畅会导致决策延迟和妥协 | 缺少涉及消费者选择任务的研究，多集中在逻辑题上；感知不畅与增强被试分析处理之间的关系存在质疑 | 目前感知不畅度对消费者选择领域的影响及其潜在机制理解缺失 | 基于直觉型线索决策即可被理解为减少分析深度，也可被理解为细致分析后的方法选择，本研究没有区分；本研究不流畅集中在视觉线索上 |

{:.academia}
| POC-RPP                                                      | IV                                         | DV                                      | Method                                                       |
| ------------------------------------------------------------ | ------------------------------------------ | --------------------------------------- | ------------------------------------------------------------ |

## UV

- 帮助 `uv [] --help`
- 更新 `pip install --upgrade uv`
- Python管理
  - 安装版本 `uv python install`
  - 列出版本 `uv python list`
  - 删除版本 `uv python uninstall`
  - 寻找版本 `uv python find`
  - 指定版本 `uv python pin`
- 脚本管理
  - 运行 `uv run`
  - 添加脚本级依赖 `uv add --script`
  - 删除脚本级依赖 `uv remove --script`
- 项目管理
  - 初始化 `uv init`
  - 添加项目依赖 `uv add`
  - 删除项目依赖 `uv remove`
  - 项目依赖与环境同步 `uv sync` (类似pip install -r)
  - 创建lockfile `uv lock`
  - 运行命令 `uv run`
  - 依赖树（显示声明的） `uv tree`
  - 打包 `uv build`
  - 发布 `uv publish` 需要配置仓库
- Tools
  - 临时应用包 `uvx / uv tool run`
    - 创建一个隔离环境，下载运行工具后清理，避免污染全局环境
    - 常用于pytest等工具
  - 安装到用户全局 `uv tool install`
  - 卸载用户全局包 `uv tool uninstall`
  - 列出用户全局包 `uv tool list`
- 环境管理
  - 创建虚拟环境 (通常uv init会自动创建，无需手动) `uv venv`
  - 安装到当前环境 `uv pip install`
  - 卸载环境包 `uv pip uninstall`
  - 显示安装包详情 `uv pip show`
  - 列出依赖 `uv pip freeze`
  - 易读列出 `up pip list`
  - 兼容性校验 `uv pip check`
  - 环境依赖树 `uv pip tree`
  - lockfile `uv pip compile`
  - 同步 `uv pip sync`

### 项目工作流

```cmd
# 初始化项目 生成配置文件
uv init
uv run main.py

# 添加依赖
uv add numpy
uv add 'pandas>=1.0.0"
uv add git+https://github.com/psf/request'
uv add -r requirements.txt -c constraints.txt
uv remove numpy
uv lock --upgrade-packages requests

# 发布
uv build
uv version 1.0.0
## 版本增量
uv version --bump [major|minor|patch|stable|aplpha|beta|rc]
uv publish

```
