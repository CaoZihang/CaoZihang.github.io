## UV

- 帮助 `uv --help`
- 更新 `pip install --upgrade uv`
- Python管理
  - 安装版本 `uv python install`
  - 列出版本 `uv python list`
  - 删除版本 `uv python uninstall`
  - 寻找版本 `uv python find`
  - 指定版本 `uv python pin`
- 脚本管理
  - 运行 `uv run`
